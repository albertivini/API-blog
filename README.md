# API Rest de um Blog

Recursos utilizados:

- Node.Js
- Express
- Validator
- JsonWebToken
- Sequelize
- SQLite


## Executar a aplicação

Para instalar os pacotes:

```
npm install
```

Para executar a API:

```
npm start
```

## Funcionalidades

- Cadastrar usuário
- Autenticar usuário
- Mostrar usuários e usuário específico
- Publicar posts
- Visualizar posts
- Visualizar feed
- Atualizar posts
- Seguir perfil
- Favoritar post
